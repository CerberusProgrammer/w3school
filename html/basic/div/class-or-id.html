Marcadores con ID y enlaces Los marcadores HTML se utilizan para permitir a los lectores saltar a partes específicas de una página web. Los marcadores pueden ser útiles si su página web es muy larga. Para hacer un marcador, primero debe crear el marcador y luego agregarle un enlace. Cuando se hace clic en el enlace, la página se desplazará a la ubicación con el marcador.
